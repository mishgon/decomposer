**`portfolio_optimized.c`** — C extension with two functions:
- `portfolio_risk_c`: Computes `sqrt(x^T * S * x)` by first computing the matrix-vector product `S*x` then the dot product with `x`. Works directly with Python lists (via `PyList_GetItem`/`PyFloat_AS_DOUBLE`) to avoid numpy conversion overhead, while also supporting numpy arrays.
- `portfolio_return_c`: Computes `weights^T * returns` as a simple dot product. Same dual-path approach.

**`portfolio_optimized.py`** — Thin Python wrapper that passes inputs directly to the C extension without any conversion, keeping overhead minimal.
